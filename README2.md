# ⚽ Primeira Liga Portuguesa — Player Comparison Tool

## 📌 Overview
This project is an interactive player comparison tool built in **Google Sheets**, designed to analyze and compare individual football player statistics from the **Primeira Liga Portuguesa**. It enables side-by-side comparison of two players across a wide range of basic and advanced metrics, with visual indicators to highlight performance differences.

👉 [**Open the Player Comparison Tool**](https://docs.google.com/spreadsheets/d/1qoU-nzAiBECSZdr9Qg7m8UaFB5PxZRcO5wefyPYUg-Y/edit?usp=sharing)

## 🔍 Key Features
- 🔎 Dynamic player selection (Player A vs Player B)
- 📊 Side-by-side statistical comparison
- 📈 Score difference indicators (▲ better / ▼ worse)
- 🖼️ Player photos and club badges
- ⚽ Basic metrics: minutes played, goals, assists, fouls
- 🧠 Advanced metrics: pass completion %, tackles + interceptions, aerial duels won %
- 🌍 Covers players from the Primeira Liga Portuguesa

## 🚀 How to Use
1. Click the link or the image above to open the Google Sheet
2. Use the **dropdown menus** to select Player A and Player B
3. The sheet will automatically update all stats and indicators
4. Green **▲** means Player A is performing better in that metric
5. Red **▼** means Player A is performing worse in that metric

## 📊 Metrics Explained

| Metric | Description |
|--------|-------------|
| Min | Total minutes played |
| Pas Comp % | Passing completion percentage |
| Tkl + Int | Tackles and interceptions combined |
| Fls | Fouls committed |
| Int | Interceptions |
| Aerial Duels Won % | Percentage of aerial duels won |
