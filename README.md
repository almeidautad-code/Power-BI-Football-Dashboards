# ⚽ Football Player Performance Dashboard

## 📌 Overview
This project is an interactive dashboard built using **Microsoft Power BI** to analyze individual football player performance across matches. It enables dynamic filtering by player and date, providing a clear view of key performance indicators and technical metrics. Data was sourced from **Wyscout**, a professional football database.

## 🔍 Key Features
- 🔎 Interactive player slicer to filter and compare individual players
- 📅 Date range filtering
- ⏱️ Minutes played per match
- 📊 KPI cards (minutes, goals, assists, cards)
- ⚔️ Offensive & defensive performance metrics
- 📈 Match-by-match performance breakdown

## 🚀 How to Use

2. Open it using **Microsoft Power BI Desktop**
