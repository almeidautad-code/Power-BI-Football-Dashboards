Football Player Performance Dashboard

Overview:

This project is an interactive dashboard built using Microsoft Power BI to analyze individual football player performance across matches.
It enables dynamic filtering by player and date, providing a clear view of key performance indicators and technical metrics.
Data was exported by Wyscout (football database)

🔍 Key Features
🔎 Player selection filter
📅 Date range filtering
⏱️ Minutes played per match
📊 KPI cards (minutes, goals, assists, cards)
⚔️ Offensive & defensive performance metrics
📈 Match-by-match performance breakdown
📁 Project Structure

images/ → Dashboard screenshots

🚀 How to Use
Download the dashboard.pbix file
Open it using Microsoft Power BI Desktop
Interact with filters and visuals
